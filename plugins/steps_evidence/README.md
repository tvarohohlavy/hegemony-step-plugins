# hegemony-steps-evidence

Assertions and diffs over evidence artifacts produced by collection steps.
Paradigm-neutral: works on artifact content regardless of how it was collected.

Handlers: `evidence.assert`, `evidence.compare`.

Namespace prefix (= entry-point name): `evidence.` — see the repo-level
`CONVENTIONS.md` for the naming rules.
